# Student CGPA API

## Objective
REST API to manage student CGPA records using in-memory JSON database.

## Routes Implemented

GET /students

GET /students/topper

GET /students/average

GET /students/count

GET /students/:id

GET /students/branch/:branchName


## Sample URL
http://localhost:5000/students

## Run Locally
npm install
npm run dev
